# 第 1 章 Cursor 簡介與安裝

## 1-2 安裝 Cursor

### 下載並安裝 Cursor

- 透過 winget 安裝的指令：

    ```
    winget show Anysphere.cursor
    ```

- 透過 scoop 安裝的指令：

    ```
    scoop bucket add extras
    scoop install cursor
    ```

- 透過 homebrew 安裝的指令：

    ```
    brew install --cask cursor
    ```

- Linux 下的安裝指令：

    ```
    sudo dpkg -i cursor-2.1-linux-x64.deb
    ```

## 1-3 牛刀小試--安裝管理 Python 環境的 uv 工具

### 指揮 Agent 幫我們安裝 uv 工具

- 輸入的提示內容：

    ```md
    用原生方式安裝 uv，不要使用 pip 方式安裝
    ```

> [!TIP]
> 萬一 Agent 無法幫你正確安裝 uv，你也可以字行在終端機執行以下指令安裝：
> - Windows 
>   ```
>   powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
>   ```
> - macOS/Linux
>   ```
>   curl -LsSf https://astral.sh/uv/install.sh | sh
>   ```

# 第 2 章 Cursor 快速入門--以設計猜數字遊戲為例

本章最後完成的[專案](https://github.com/FlagTech/F6722_ch02_1A2B_cli)

## 2-1 跟 AI 交談聊出專案--設計文字模式的猜數字遊戲

### 用交談方式說明需求

- 輸入的提示內容：

    ```md
    設計文字模式 1A2B 猜數字遊戲；

    - 輸入猜測數字後不要換行，把幾 A 幾 B 的結果顯示在猜測數字的右側

    - 單次遊戲最多可猜 10 次，超過就算失敗

    - 單次遊戲結束後要顯示成功或失敗的訊息，並詢問是否重玩？
    ```

### 修正 AI 的工作規劃

- 輸入的提示內容：

    ```md
    我用 uv 管理 Python 專案的環境，請用 uv 將專案初始成使用 Python 3.13
    ```

## 2-2 與 AI 協作的除錯方式

### 把錯誤描述清楚的除錯法

- 輸入的提示內容：

    ```
    現在輸入猜測數字後會先換行，才在新的一行顯示猜測的數字與幾 A 幾 B 的結果，請修正為輸入猜測數字後不換行，直接在猜測的數字右側顯示幾 A 幾 B 的結果
    ```

### 幫遊戲上色

- 輸入的提示內容：

    ```
    我想要讓幾 A 幾 B 用不同顏色顯示
    ```

### 用 @ 引用功能讓 Agent 看問題畫面

- 輸入的提示內容：

    ```
    從畫面中可以看到遊戲標題右側的框框有破洞    
    ```

## 2-4 請 AI 寫文件

### 讓 Agent 根據實作成果撰寫規格書

- 輸入的提示內容

    ```md
    把目前成果整理成正式的規格書，儲存在 spec.md 中
    ```

### 讓 Agent 根據規格書回頭撰寫產品需求書

- 輸入的提示內容

    ```md
    從 @spec.md 幫我整理一份產品需求書，儲存在 prd.md 中
    ```
